# 🌿 Shanti — Heal & Meditate

## 📌 Overview
Shanti is a modern wellness web application designed to help users relax, meditate, and improve mental well-being through breathwork, sound therapy, guided meditation, and affirmations.

## ✨ Features
- 🧘 Breathwork patterns (Box, 4-7-8, Equal breathing)
- 🔊 Real-time sound generation (Web Audio API)
- ⏱ Guided meditation sessions with timers
- ✨ Daily affirmations + custom affirmations
- 📊 User dashboard (streaks, sessions, minutes)
- 🔐 Authentication system (localStorage-based)
- 💬 AI chat assistant (UI ready, API integration pending)
- ⭐ Feedback system

## 🛠 Tech Stack
- HTML5
- CSS3 (Modern UI with variables & animations)
- JavaScript (Vanilla JS)
- Web Audio API
- LocalStorage (for user data)
